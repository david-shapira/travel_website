# NepVenture – אתר טרקים לנפאל (HTML + CSS)

## תאור כללי
הפרויקט הוא אתר סטטי העוסק בטרקים בנפאל. האתר כולל ארבעה עמודים שונים, תפריט ניווט אחיד, עיצוב רספונסיבי, שימוש ב־Flexbox ועיצוב מלא באמצעות CSS בלבד, ללא JavaScript.

## מבנה הפרויקט
/
├── index.html        – דף הבית  
├── routes.html       – דף המסלולים  
├── gallery.html      – גלריית תמונות  
├── about.html        – עמוד אודות  
├── styles.css        – קובץ עיצוב ראשי  
└── images/           – תיקיית תמונות (לוגו, בנאר, מסלולים וכו)

## מטרות הפרויקט
1. בניית אתר מלא הכולל ניווט תקין בין ארבעה עמודים.  
2. שימוש ב־Flexbox לבניית גלריה/כרטיסיות/פריסה.  
3. כתיבת CSS מסודר עם שמות מחלקות עקביים.  
4. יצירת עיצוב רספונסיבי באמצעות media query.  
5. עמידה בכל חוקים: אין JavaScript, אין ספריות חיצוניות.

## תכונות בולטות
- Header ו־Footer אחידים בכל העמודים.  
- עיצוב נקי ורספונסיבי.  
- כרטיסיות מסלולי טרקים מעוצבות באמצעות Flexbox.  
- גלריית תמונות רספונסיבית.  
- מבנה קוד מסודר ונקי למניעת שכפולים.

## איך להריץ את האתר
1. שמים את כל הקבצים בתיקייה אחת.  
2. פותחים את `index.html` בדפדפן.  
3. עוברים בין העמודים דרך התפריט העליון.
